2) JavaScript se puede integrar a HTML de dos formas:

La primer forma, y la más utilizada, es creando un archivo "Script.JS" aparte y conectarlo con nuestro Index.HTML utilizando las etiquetas "Script", dentro del Body o del Head, con el atributo "src" para poner el nombre del archivo.
Y la otra forma de integrar a JavaScript a HTML es escribir el código JS dentro de las etiquetas "Script", que pueden estar en el Body o en el Head.

3) El manejo de Variables en JS es el siguiente: A la hora de escribir el nombre de la variable hay que tener en cuenta las mayúsculas y las minúsculas, ya que JS las distinque. Y pueden ser declaradas usando las palabras claves var, let o const, escritas en minúsculas. El valor de una variable declarada con var y let se puede sobrescribir, mientras que const no. Y let y const tienen alcance de bloque, estando limitadas por las llaves "{}".

Los Tipos de Datos en JS, según el último estándar de ECMAScript, son ocho:

  • Booleano: Su valor puede ser true o false.
  • BigInt: Es un número entero con precisión arbitraria.
  • String (Cadena): Es una secuencia de caracteres que forman un texto.
  • Number (Número): Puede ser entero o decimal.
  • Null (Nulo): Es una palabra especial que denota un valor nulo.
  • Undefined (Indefinido): Es una propiedad de alto nivel cuyo valor no está definido.
  • Symbol (Símbolo): Crea identificadores únicos para los Objetos.
  • Object (Objeto): Son contenedores de colecciones de datos.

Las Expresiones Regulares en JS se pueden crear de forma literal, encerrando la expresión entre barras "//", o llamandola usando "RegExp". Y se usan con los siguientes métodos:
  • exec(): Ejecuta una búsqueda por una coincidencia en una cadena. Devuelve un arreglo de información o null.
  • test(): Prueba una coincidencia en una cadena. Devuelve true o false.
  • match(): Devuelve un arreglo que contiene todas las coincidencias, de lo contrario devuelve null.
  • matchAll(): Devuelve un iterador que contiene todas las coincidencias.
  • search(): Prueba una coincidencia en una cadena. Devuelve el índice de la coincidencia, pero si la busqueda falla, devuelve -1.
  • replace(): Ejecuta una búsqueda por una coincidencia en una cadena, remplazando la subcadena coincidente con una de reemplazo.
  • replaceAll(): Ejecuta una búsqueda de todas las coincidencias en una cadena, remplazando la subcadena coincidente con una de reemplazo.
  • split(): Utiliza una cadena fija o una expresión regular para dividir una cadena en un arreglo de subcadenas.
